N = Anzahl Teilnehmer
M = Anzahl Partner
K = Anzahl Partner der Partner

```
      ---> Gewonnen
     /
    / 1/N            ---> Gewonnen
   /                /
  /                / 1/K
 /   M/N          /
* ---------------*
 \                \
  \                \ (K-1)/K
   \                \
    \ (N-(M+1))/N    ---> Verloren
     \
      ---> Verloren
```
